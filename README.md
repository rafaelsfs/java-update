# Script to install or update Java manually Ubuntu

## Install

``` bash
git clone https://github.com/rafaelsfs/java-update.git
```
``` bash
cd /java-update
```
``` bash
chmod + x java-update.sh
```
``` bash
./java-update.sh
```

## Uninstall
``` bash
cd /java-update
```
``` bash
chmod + x java-update.sh
```
``` bash
./uninstall-java.sh
```
